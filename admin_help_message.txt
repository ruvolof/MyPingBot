Commands:

/help Show this help message.
/stats Show bot statistics.
/broadcast Send a message to all users.
/manualCheck Run the monitor check routine manually.
/dumpStatus Dump on console the servers_list object.
/listUsers Show list of active users.
/listServers USERNAME Shows servers monitored by USERNAME.
/removeUser USERNAME Removes the user.
/removeHostFromUsername USERNAME HOST Removes HOST from USERNAME monitor list.
/adminmode Exits admin mode.
